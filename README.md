# react-native-chainway-rfid

Chainway C72 RFID Reader Lib

## Installation

```sh
npm install react-native-chainway-rfid
```

## Usage

```js
import { multiply } from "react-native-chainway-rfid";

// ...

const result = await multiply(3, 7);
```
